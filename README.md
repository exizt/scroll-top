# scroll-top

스크롤을 상위로 올리는 화살표를 화면 우측 하단에 만들어주는 스크립트.

# 데모

# 사용법
## 모던 브라우저 
스크립트 로드
```html
<script src="scroll-top.js"></script>
```


## 예전 브라우저(IE11 등)
스크립트 로드
```html
<script src="scroll-top.legacy.js"></script>
```

데모 링크 : https://exizt.github.io/scroll-top/test/legacy-test.html



# 브라우저 지원
